# 📊 FalteiSenai

Projeto desenvolvido para visualizar e acompanhar faltas no SENAI de forma simples, rápida e acessível.

## 🚀 Acesse o projeto
👉 https://victorbarretoandrade.github.io/FalteiSenai/

---

## 📌 Sobre o projeto

O **FalteiSenai** é uma aplicação web que permite ao usuário verificar e acompanhar suas faltas de maneira prática, sem precisar navegar por sistemas complexos.

O objetivo é facilitar o acesso à informação e melhorar a organização acadêmica.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages (deploy)

---

## 📷 Funcionalidades

- ✅ Interface simples e intuitiva  
- ✅ Visualização de faltas  
- ✅ Acesso rápido via navegador  
- ✅ Responsivo (funciona em diferentes dispositivos)
